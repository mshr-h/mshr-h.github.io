Ubuntuで最新版Node.jsをインストールする

    "Masahiro Hiramori" included in Tech
 2020-03-07  219 words   One minute 
Contents

aptなどのパッケージマネージャでnode.jsをインストールすると、古いバージョンが導入されることがある。n packageを使うことで、簡単にnode.jsの最新版を導入できる。

手順

  1. node.jsとnpmを導入

    1

    $ sudo apt install -y nodejs npm

  2. n packageを導入

    1

    $ sudo npm install -g n

  3. n packageを使い、nodeの最新版を導入

    1

    $ sudo n stable

  4. 最後に、apt-getで導入したnode.jsとnpmを削除

    1

    $ sudo apt purge -y nodejs npm

  5. 正しく導入されていることを確認

    1
2

    $ node -v
v12.16.1
