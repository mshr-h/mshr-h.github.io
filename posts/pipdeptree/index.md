# pipdeptreeでPythonパッケージの依存ツリーを見る


とあるPythonパッケージが依存しているパッケージを表示するには、`pipdeptree`が便利。
オプションの指定なしで実行すると、全パッケージの依存リストが表示される。
`-p package_name`オプションで特定のパッケージに絞ることができる。

## tensorflowの依存一覧を表示する例

```bash
$ pipdeptree -p tensorflow
```
